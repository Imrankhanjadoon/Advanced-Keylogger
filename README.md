\# 🔒 Educational Keylogger



A simple yet professional Python keylogger built for \*\*educational and portfolio purposes only\*\*.



\## 🎯 Features

\- Real-time keystroke capturing (letters + special keys)

\- Timestamp in every log

\- Clean Tkinter GUI (Start / Stop / View Logs)

\- Cross-platform (Windows, Linux, macOS)

\- Logs saved in `keylogs.txt`



\## 🛠️ Technologies Used

\- Python 3

\- pynput (keyboard listener)

\- Tkinter (GUI)



\## 📸 How to Run

1\. `pip install -r requirements.txt`

2\. `python keylogger.py`

3\. Click \*\*START Logging\*\*



\## 📚 What I Learned

\- Keyboard event hooking

\- Threading with GUI

\- Proper logging practices

\- Ethical considerations in cybersecurity



\## 🛡️ Defensive Side 

\*\*How to detect/prevent such keyloggers:\*\*

\- Check Task Manager / Activity Monitor for suspicious Python processes

\- Use antivirus (Windows Defender, Malwarebytes)

\- Monitor keyboard hooks with tools like Process Explorer

\- Enable "Core Isolation" in Windows

\- Never download unknown .py or .exe files



\*\*Note:\*\* This project was created only to understand offensive techniques so we can defend better.
